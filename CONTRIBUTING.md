# Contributing

Thanks for helping improve ToolCapsule.

## Development

```bash
corepack enable
pnpm install
pnpm run ci
```

## Pull requests

Please include:

- a clear summary;
- tests for behavior changes;
- documentation updates for user-facing changes;
- screenshots or CLI output for UX changes.

## MCP profiles

When contributing a new MCP profile, avoid committing secrets or private URLs. Use placeholders and document required environment variables.
